# Contributing to Py-to-EXE Pro

First off, thank you for considering contributing to this tool! 

### How Can I Contribute?

* **Reporting Bugs:** Open an issue with a clear description and steps to reproduce.
* **Suggesting Enhancements:** We are always looking for new features (e.g., PyArmor integration).
* **Pull Requests:** * Maintain the modular structure.
    * Ensure all code is commented.
    * Test your build on Windows 10/11 before submitting.

### Style Guide
We follow [PEP 8](https://www.python.org/dev/peps/pep-0008/) for Python code. Please ensure your indentation is consistent (4 spaces).
